# 24. How could you better understand the user in future design tasks? Discuss. [6]

1. become a user
    + cheap and easy
    + lots of ideas
    - mistakes and false observations
    - preconceptions
    - physical differences
    - already understand the product

2. observe users
    + impartial
    + experts show shortcuts
    + beginners show learning curve
    - harder to do

3. run experiments
    e.g.
    storyboards
    mechanical turk / wizard of oz
    pinnochio

4. translate the above into outcomes
